Funzione periodica

In matematica, a livello intuitivo, per funzione periodica si intende una funzione che assume valori che si ripetono esattamente a intervalli regolari.

Definizione[modifica | modifica wikitesto]

Una funzione f : A → B {\displaystyle f\colon A\to B} definita su un gruppo abeliano A {\displaystyle A} è periodica di periodo t {\displaystyle t} , con t ∈ A {\displaystyle t\in A} , se f ( a + t ) = f ( a ) {\displaystyle f(a+t)=f(a)} per ogni a ∈ A {\displaystyle a\in A} .

Funzioni di variabile reale[modifica | modifica wikitesto]

Le funzioni periodiche più note sono le funzioni reali di variabile reale. Formalmente, una funzione reale f : A → B {\displaystyle f\colon A\to B} si dice periodica di periodo t {\displaystyle t} se esiste un numero reale t ∈ R {\displaystyle t\in \mathbb {R} } tale che:

  • il dominio A {\displaystyle A} è invariante per traslazione di t {\displaystyle t} , ovvero A + t = { a + t ∣ a ∈ A } = A {\displaystyle A+t=\{a+t\mid a\in A\}=A} ;
  • la funzione f {\displaystyle f} è invariante per traslazione di t {\displaystyle t} , ovvero per ogni a ∈ A {\displaystyle a\in A} si ha f ( a + t ) = f ( a ) {\displaystyle f(a+t)=f(a)} .

Moduli[modifica | modifica wikitesto]

Se f {\displaystyle f} è periodica di periodo t 1 {\displaystyle t_{1}} ed è periodica di periodo t 2 {\displaystyle t_{2}} , allora è periodica di ogni periodo

t ∈ t 1 Z + t 2 Z = { m t 1 + n t 2 ∣ m , n ∈ Z } {\displaystyle t\in t_{1}\mathbb {Z} +t_{2}\mathbb {Z} =\{mt_{1}+nt_{2}\mid m,n\in \mathbb {Z} \}} .

L'insieme T f {\displaystyle {\mathcal {T}}_{f}} dei periodi t {\displaystyle t} di f {\displaystyle f} è quindi uno Z {\displaystyle \mathbb {Z} } -modulo.

  • Se T f = { 0 } {\displaystyle {\mathcal {T}}_{f}=\{0\}} , ovvero se f {\displaystyle f} ha il solo periodo 0 {\displaystyle 0} , allora f {\displaystyle f} è detta aperiodica.
  • Se T f {\displaystyle {\mathcal {T}}_{f}} è un modulo libero di dimensione 1 {\displaystyle 1} , ovvero se T f = t Z {\displaystyle {\mathcal {T}}_{f}=t\mathbb {Z} } con t > 0 {\displaystyle t>0} , ovvero se esiste un minimo tra i periodi t > 0 {\displaystyle t>0} , allora f {\displaystyle f} è detta periodica di periodo minimo t {\displaystyle t} , o periodica di periodo t {\displaystyle t} in senso stretto.
  • Il modulo T f {\displaystyle {\mathcal {T}}_{f}} non è necessariamente libero di dimensione 0 {\displaystyle 0} o 1 {\displaystyle 1} , ovvero potrebbe non esistere un minimo periodo strettamente positivo; ad esempio, la funzione di Dirichlet ha T f = Q {\displaystyle {\mathcal {T}}_{f}=\mathbb {Q} } e non è né aperiodica né periodica in senso stretto.

Domini limitati[modifica | modifica wikitesto]

Da ogni funzione a valori reali definita su un dominio limitato si può definire una funzione periodica, di periodo maggiore o uguale all'ampiezza del dominio. Ad esempio, la funzione identità ristretta all'intervallo [ 0 , 1 ) {\displaystyle [0,1)} ,

f : [ 0 , 1 [ → R x ↦ x , {\displaystyle {\begin{aligned}f\colon [0,1[&\to \mathbb {R} \\x&\mapsto x,\end{aligned}}}

definisce una funzione periodica di periodo 1 definita su tutti i reali: la parte frazionaria

f ~ : R → R x ↦ x − [ x ] . {\displaystyle {\begin{aligned}{\tilde {f}}\colon \mathbb {R} &\to \mathbb {R} \\x&\mapsto x-[x].\end{aligned}}}

Esempi[modifica | modifica wikitesto]

  • Le funzioni trigonometriche seno e coseno sono periodiche di periodo minimo 2 π {\displaystyle 2\pi } .
  • Sono quindi automaticamente periodiche le funzioni:
    • tan ⁡ ( x ) = sin ⁡ ( x ) cos ⁡ ( x ) {\displaystyle \tan(x)={\frac {\sin(x)}{\cos(x)}}} e cot ⁡ ( x ) = cos ⁡ ( x ) sin ⁡ ( x ) {\displaystyle \cot(x)={\frac {\cos(x)}{\sin(x)}}} , che hanno periodo minimo π {\displaystyle \pi } ;
    • sec ⁡ ( x ) = 1 cos ⁡ ( x ) {\displaystyle \sec(x)={\frac {1}{\cos(x)}}} e csc ⁡ ( x ) = 1 sin ⁡ ( x ) {\displaystyle \csc(x)={\frac {1}{\sin(x)}}} , che hanno periodo minimo 2 π {\displaystyle 2\pi } .

Funzioni doppiamente periodiche[modifica | modifica wikitesto]

Una funzione può ammettere due o più periodi non commensurabili (la definizione dipende dalle caratteristiche che si richiedono al dominio).

Ad esempio, una funzione ellittica è una funzione doppiamente periodica:

è definita dall'insieme dei numeri complessi in sé, f : C → C {\displaystyle f\colon \mathbb {C} \to \mathbb {C} } ;
è periodica rispetto a due periodi, ω 1 , ω 2 ∈ C ∗ ,   ∀ z ∈ C   f ( z + ω 1 ) = f ( z + ω 2 ) = f ( z ) {\displaystyle \omega _{1},\omega _{2}\in \mathbb {C} ^{\ast },\ \forall z\in \mathbb {C} \ f(z+\omega _{1})=f(z+\omega _{2})=f(z)} ;
questi due periodi sono "incommensurabili", ω 1 / ω 2 ∉ R . {\displaystyle \omega _{1}/\omega _{2}\not \in \mathbb {R} .}
